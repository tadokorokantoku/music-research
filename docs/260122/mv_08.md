## 📋 採用基準

### 2軸スコアリング

このレポートは **MVスコア** と **日本スコア** の2軸でフィルタリングしています。

#### MVスコア (0-100点)

MVとしての品質を評価します（国は考慮しない）。

- **+20点**: タイトルに "MV" / "Music Video" / "ミュージックビデオ"
- **+15点**: 制作クレジット記載（監督、Director等）
- **+10点**: タイトルに "Official" または公式チャンネル
- **+15点**: 標準的な長さ（3-5分）
- **-40点**: カバー・歌ってみた等
- **-100点**: Shorts

#### 日本スコア (0-100点)

日本の動画かどうかを評価します。

- **100点**: チャンネル国=JP（即確定）
- **+80点**: defaultLanguage=ja
- **+30点**: タイトルに日本語
- **+10点**: 日本特有の記号【】
- **-50点**: チャンネル国が他国

```
🌟 確実な日本MV: MVスコア≥70 AND 日本スコア=100
```
